# AI Resume Generator

## 🚀 Overview
This project generates professional resumes using AI based on user input. It uses prompt engineering techniques to create structured and role-specific resumes.

## ✨ Features
- Generates clean and structured resumes  
- Role-based customization  
- Simple and easy to use  

## 🧠 Prompt Engineering Used
- Role-based prompting  
- Structured output formatting  
- Iterative refinement  

## 🛠 Tech Stack
- Python  
- OpenAI API  

## ▶️ How to Run
1. Install Python  
2. Set your API key:
   export OPENAI_API_KEY=your_key_here  
3. Run:
   python app.py  

## 📌 Example
Input: Skills + Role  
Output: Professional resume
